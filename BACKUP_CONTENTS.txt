RESTORE INSTRUCTIONS:
--------------------

To restore this backup:

1. Extract the zip file:
   unzip long-context-bench-backup-20251106_141435.zip

2. The extracted files will be in output/ and data/ directories

3. To view the web dashboard:
   cd output/web
   npm install
   npm start
   # Open http://localhost:3000

4. To run additional benchmarks:
   long-context-bench edit data/samples/v0 --runner <runner> --model <model>

5. To judge new runs:
   long-context-bench judge --edit-run-ids <run_id>
