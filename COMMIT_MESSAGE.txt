refactor: Remove head-to-head evaluation, keep cross-agent analysis only

This commit removes all head-to-head (pairwise agent-as-judge) evaluation
functionality, simplifying the benchmark to focus exclusively on cross-agent
LLM-based comparative evaluation.

## Motivation

The head-to-head evaluation approach had several limitations:
- More complex to implement and maintain (~2,000 lines of code)
- Less powerful than LLM-based comparative analysis
- Redundant with cross-agent analysis which provides richer insights
- Created confusion with two evaluation paradigms

Cross-agent analysis is superior because:
- Provides detailed comparative analysis and reasoning
- More transparent and verifiable
- Single, consistent evaluation method
- Easier to understand and maintain
